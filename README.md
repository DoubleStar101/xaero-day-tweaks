# Xaero Day Tweaks

A lightweight Fabric mod that synchronizes **Xaero's Minimap** day counter with Minecraft's vanilla day counter (found in the f3 menu).

By default, Xaero's Minimap starts counting at day 1, while vanilla Minecraft starts counting at day 0. This mod changes Xaero's display to match vanilla without modifying Xaero's original JAR.

## Features

* Synchronizes Xaero's day counter with vanilla Minecraft
* Adds the ability to add a custom offset to the Xaero's minimap day counter

## Compatibility

| Component       | Version      |
| --------------- | ------------ |
| Minecraft       | 26.2         |
| Fabric Loader   | 0.19.3+      |
| Fabric API      | 0.153.0+26.2 |
| Xaero's Minimap | 26.1.3       |

## Dependencies

- Fabric Loader.
- Fabric API.
- Xaero's Minimap.
